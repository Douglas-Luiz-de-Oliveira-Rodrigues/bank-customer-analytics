# 📊 Bank Customer Analytics

Projeto prático de análise de dados simulando um cenário bancário, com foco em atividades comuns de um estágio na área de dados.

---

## 🎯 Objetivo

Desenvolver um fluxo simples de análise de dados, incluindo:
- manipulação de dados
- criação de variáveis (feature engineering)
- consultas SQL
- construção de modelo de machine learning
- criação de dashboard

---

## 🛠️ Tecnologias utilizadas

- Python (Pandas, Scikit-learn)
- SQL
- Power BI
- Git/GitHub

---

## 📁 Estrutura do projeto

bank-customer-analytics/
│
├── data/ # Base de dados
├── sql/ # Consultas SQL
├── notebook/ # Análise em Python
├── dashboard/ # Dashboard (Power BI ou imagem)
└── README.md


---

## 📌 Etapas do projeto

1. Coleta e organização dos dados  
2. Análise exploratória  
3. Criação de variáveis relevantes  
4. Modelagem preditiva simples  
5. Consultas SQL para apoio à análise  
6. Visualização dos dados em dashboard  

---

## 📊 Resultados esperados

- Identificação de padrões de clientes  
- Análise de comportamento (ativos vs inativos)  
- Modelo simples de previsão  
- Visualização clara dos dados  

---

## 🚀 Objetivo profissional

Este projeto foi desenvolvido com o objetivo de praticar habilidades essenciais para atuação na área de dados, especialmente em contextos bancários e financeiros.
